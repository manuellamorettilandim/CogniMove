# CogniMove

Dashboard moderno para monitoramento inteligente do transito com dados simulados e arquitetura preparada para futura API REST.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui style components
- Lucide Icons
- Framer Motion
- Recharts
- React Hook Form

## Experiencias incluidas

- Abertura animada de Smart City com deteccoes simuladas e barra de progresso.
- Dashboard operacional inspirado em centrais inteligentes de transito.
- Experiencia Interativa em 6 etapas para apresentacao na FECART.
- Centro de Operacoes com mapa clicavel, painel lateral e Desafio Smart City.

## Observacao

Nao ha backend neste momento. Todos os dados ficam em `mock/traffic-data.ts` e podem ser substituidos futuramente por services REST.
